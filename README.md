# nasl
Nasl is Not A Sixteen-bit Library

## Requirements

* CMake 2.8+
* Glfw3
* Glew (Optionally)

## Using

* MY-BASIC
* Nuklear
* stb_image

## How to build

    mkdir build
    cd build
    ccmake ..
    make -j4
