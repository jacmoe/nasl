# nasl
Nasl is Not A Sixteen-bit Library

## Requirements

* CMake 2.8+
* [Glfw3][glfw]
* Glew (Optionally)

## Using

* [MY-BASIC][mybasic]
* [Nuklear][nuklear]
* [stb_image][stbimage]

## How to build

    mkdir build
    cd build
    ccmake ..
    make -j4
